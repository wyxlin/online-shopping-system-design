# Documentation

This folder contains design notes and learning documents for the online shopping system design study project.

The documents in this folder focus on high-level architecture, quality attributes, and design trade-offs. They are intended for learning and documentation purposes rather than describing a full production implementation.

## Contents

- `architecture.md` – high-level system structure and major components
- `quality-attributes.md` – example non-functional requirements and scenario-based analysis
- `trade-offs.md` – example design decisions and their trade-offs
